# Random Joke API Script

A simple Python script that fetches a random joke from the Official Joke API and prints the response status code and joke data.

## What It Does

- Sends a GET request to:
  https://official-joke-api.appspot.com/random_joke
- Prints the HTTP status code
- Prints the returned JSON joke

## Requirements

- Python 3
- `requests` library

Install requests if needed:

pip install requests
