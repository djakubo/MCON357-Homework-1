# Random Joke API Script

This script calls the Official Joke API:

https://official-joke-api.appspot.com/random_joke

## API Called
Official Joke API (public REST API)

## Access Method Used
HTTP GET request using Python's `requests` library.

## What It Does
- Sends a GET request to the API endpoint
- Prints the HTTP status code
- Prints the returned JSON response

## Requirements
- Python 3
- requests library (`pip install requests`)
